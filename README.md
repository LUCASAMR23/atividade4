## Sobre o projeto

Este site permite cadastrar e visualizar jogos da Mega-Sena. O usuário informa seis números entre 1 e 60, e o sistema valida se a quantidade está correta, se não existem números repetidos e se todos estão dentro do intervalo permitido.

Depois da validação, o jogo é salvo no banco de dados PostgreSQL e exibido na tela em formato de bolinhas, facilitando a visualização das dezenas cadastradas.
